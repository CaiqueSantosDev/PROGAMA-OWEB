O que é HTML?

HTML significa HyperText Markup Language, que em português seria algo como "Linguagem de Marcação de Hipertexto". Ele é a base de qualquer página web e serve para estruturar o conteúdo dela. Ou seja, o HTML não é responsável por deixar o site bonito ou interativo (isso é feito com CSS e JavaScript), mas sim por organizar o conteúdo de maneira lógica.
